Pour assurer la migration de notre application:
Nous commencons par l'extraction du modèle de l'application depuis de code source.
Puis on passe à la représentation des interfaces utilisateur, pour cela nous devons extraire, depuis le modèle de code source, un modèle d'interface graphique.
Nous générons l'interface graphique dans l'application cible (le code sera écrit en Angular).
